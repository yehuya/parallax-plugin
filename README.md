# parallax plugin (JS)
## background
Parallax for background

<code>
  $('#elem').parallax().background(distance, transition);
</code>

## box
Element movment

<code>
  $('#elem').parallax().box(distance, transition);
</code>

### @param
* @distance = Number (0-50) 
* @transition = Number (0-99999999) time for transition css - (OPTIONAL)
